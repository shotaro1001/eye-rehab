## 👁 アイリハビリ

眼球運動障害のある方向けの眼球運動トレーニングWebアプリです。画面上を移動するターゲットを目で追い、Webカメラの虹彩検出(MediaPipe Face Landmarker)で「追えているか」をおおまかに評価します。

**▶ アプリを開く: https://shotaro1001.github.io/eye-rehab/**

ブラウザだけで動作します。インストール不要・カメラ映像は端末内で処理され、どこにも送信・保存されません。

### 訓練モード(6種)

- **追従(スムーズパシュート)** — 水平・垂直・斜め・ジグザグ(反射)・円・8の字。速さ3段階+「変化(緩急)」。ターゲットが時々消える「予測追従」オプション付き
- **サッケード** — 水平/垂直/斜め2点・ランダム。通常のほか、印と反対側を見る**アンチサッケード**、一瞬光った場所を覚えて合図後に見る**記憶誘導サッケード**。数字読み上げオプション付き
- **固視** — 中央を見つめ続けて維持率を計測。周辺の妨害刺激につられると減点のオプションあり
- **視覚探索** — たくさんの「●」から1つの「★」を探してタップ(カメラ不要)
- **読みスキャン** — 読書のように左→右・上の行→下の行へ順番に光る場所を目で追う
- **VOR(前庭動眼反射)** — 左右/上下の頭振り、ターゲットが頭と逆に動く「×2」に対応

### 共通機能

- 訓練前の9点キャリブレーション(品質も記録)。カメラなしの「訓練のみ」モードにも対応
- スコアは各自のブラウザ内(localStorage)に保存され、記録タブで推移グラフを表示
- メニュー(処方プログラム)機能: 複数の訓練を保存して休憩をはさみながら連続実行
- 絵文字ターゲット・コンボ表示、難易度アップの自動提案

### 使い方のポイント

- 画面から **50〜60cm** 離れ、顔が明るく映る場所で行う
- 頭は動かさず**目だけで**追うのが基本(VORを除く)
- 初回はカメラ許可を「許可」し、検出モデルの読み込みを待つ(数十秒かかることがあります)

> ⚠️ 本アプリは眼球運動訓練の補助を目的としたものであり、医療機器ではありません。診断・治療の判断には使用できません。Webカメラによる視線検出の精度は視角2〜4度程度の「おおまかな評価」です。めまい・吐き気・強い眼精疲労を感じたらすぐに中止し、訓練の種類・速度・時間は必ず医師・視能訓練士・リハビリ専門職の指導に従ってください。

## 🧠 注意カレッジ

注意機能(持続性・選択性・分配性・転換性)と反応速度を鍛える12種のミニゲーム集です。カメラ不要、タップ/クリックだけで遊べます。

**▶ アプリを開く: https://shotaro1001.github.io/eye-rehab/attention.html**

### ゲーム一覧

- **瞬間視・判断力** — フラッシュカウント(一瞬の●の数)、どっちが多い?(左右のドット比較)、左右たしざん(一瞬の2数字の合計)
- **反応・抑制** — モグラたたき(単純反応時間)、青はGO・赤はダメ(Go/No-Go課題)
- **選択性注意** — いろよみクイズ(ストループ課題)、仲間はずれ探し(視覚探索)、まんなかの矢印(フランカー課題)
- **記憶・切り替え** — 光ったのはどこ?(Corsiブロック風の位置記憶スパン)、順番タッチ(Trail Making風、数字/かな交互切り替えあり)
- **持続性注意** — さっきと同じ?(1-back/2-backワーキングメモリ)、見張り番(CPT風の長時間持続的注意、前半/後半の見逃し率も記録)

### 共通機能

- 各ゲームとも本番前に非採点の練習を実施
- 大きなボタン配置で、視線入力・スイッチ入力を想定した**dwell選択(注視滞留での選択)**に対応(設定でON/OFF)
- 反応が遅くても待つ「時間制限なしモード」を搭載(運動麻痺等への配慮)
- レベル(やさしい/ふつう/むずかしい)ごとにスコアを記録し、前回・自己ベストとの比較、3回連続80点以上での難易度アップ提案
- 記録タブでゲーム別のスコア推移グラフを表示

> ⚠️ 本アプリは注意機能のトレーニング・セルフチェックを目的としたものであり、診断には使用できません。疲れを感じたら中止してください。

## 技術構成

- HTML / CSS / JavaScript の単一ファイル(アプリごとにビルド不要)
- アイリハビリの虹彩検出: [MediaPipe Face Landmarker](https://ai.google.dev/edge/mediapipe/solutions/vision/face_landmarker)(CDNから読み込むためインターネット接続が必要)
- データ保存: `localStorage`(外部送信なし、アプリごとに別々に保存されます)
